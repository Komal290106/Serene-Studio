# **Serene Studio – Luxury Fashion Accessories Store**

Serene Studio is a modern and elegant ecommerce web application designed for exploring and purchasing premium fashion accessories.  
Built with **React**, **Vite**, **Tailwind CSS**, and **React Router**, the project emphasizes a refined UI, smooth navigation, and a high-end shopping experience.



## **Features**

- Fully responsive luxury-themed interface  
- Product listings with detailed descriptions  
- Wishlist management  
- Add-to-cart functionality  
- Smooth and structured routing using React Router  
- Clean, minimal, and scalable UI system  
- Organized folder structure suitable for expansion  



## **Tech Stack**

- **React**  
- **Vite**  
- **CSS**
- **Javascript**
- **React Router**



## **Installation**

### **1. Clone the repository**

```bash
git clone https://github.com/your-username/serene-studio.git

2. Navigate into the project directory
cd serene-studio

3. Install dependencies
npm install

4. Start the development server
npm run dev
```
## **Project Structure**
```bash
src/
│── components/
│── pages/
│── context/
│── assets/
│── styles/
│── App.jsx
│── main.jsx
│── router/
```

## **How It Works**
- Routing is controlled through React Router.
- Styling is built with Tailwind CSS and supplemental custom CSS.
- Wishlist and cart operations use React state and context.
- Product data can be served locally or through an external API/JSON source.
